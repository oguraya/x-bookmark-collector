# X Bookmark URL Collector

X（Twitter）のブックマークページからツイートURL・メディア・引用ポスト情報を自動収集するChrome拡張機能です。

## インストール方法

1. `chrome://extensions/` を開く
2. 右上の **「デベロッパーモード」** をONにする
3. **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. ダウンロードして解凍したこのフォルダを選択

## 使い方

1. **https://x.com/i/bookmarks** を開く
2. 拡張機能アイコンをクリックしてポップアップを表示
3. **「▶ 収集開始」** ボタンを押す → 自動スクロールでブックマークを収集
4. ページ末端に到達すると **自動でTSVファイルがダウンロード** されます
5. 途中で停止したい場合は **「■ 停止」** ボタンを押し、手動で **「📊 TSVダウンロード」** できます

## 出力フォーマット（TSV）

ファイル名: `x_bookmarks_YYYY-MM-DD.tsv`

| カラム | 内容 |
|--------|------|
| `url` | ツイートのURL（`https://x.com/user/status/id`） |
| `author` | 投稿者のユーザー名（`@username`） |
| `date` | 投稿日時（ISO 8601） |
| `text` | ツイート本文（先頭200文字） |
| `media_urls` | 画像URL（スペース区切り、最大4枚、`name=orig` の元画像サイズ） |
| `has_video` | 動画/GIFを含む場合 `true` |
| `quoted_url` | 引用元ツイートのURL |
| `quoted_author` | 引用元の投稿者（`@username`） |
| `quoted_text` | 引用元の本文（先頭200文字） |
| `tags` | 自動分類タグ（カンマ区切り） |

## 自動タグ分類

各ブックマークに以下のタグが自動付与されます（複数該当する場合はカンマ区切り）。

| タグ | 条件 |
|------|------|
| `image` | 画像を含む |
| `video` | 動画またはGIFを含む |
| `quote` | 引用ツイートを含む |
| `link` | 外部リンク（リンクカード）を含む |
| `thread` | スレッドの一部 |
| `long_text` | 「さらに表示」がある長文ツイート |
| `text_only` | 上記のいずれにも該当しないテキストのみのツイート |

ポップアップ上でもタグの分布サマリーをリアルタイム表示します。

## 注意事項

- X のブックマークページ（`x.com/i/bookmarks`）以外では動作しません
- 収集データはブラウザのメモリ上にのみ保持されます（ページをリロードするとクリアされます）
- 動画の直リンク（mp4等）はストリーミング配信のため取得できません。`has_video` フラグで動画の有無を判定し、ツイートURLから別途取得してください
- X の DOM 構造変更により動作しなくなる場合があります
