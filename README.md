# X Bookmark URL Collector

X（Twitter）のブックマークページからツイートURLを自動収集するChrome拡張機能です。

## インストール方法

1. `chrome://extensions/` を開く
2. 右上の **「デベロッパーモード」** をONにする
3. **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. ダウンロードして解凍したこのフォルダを選択

## 使い方

1. **https://x.com/i/bookmarks** を開く
2. 拡張機能アイコンをクリックしてポップアップを表示
3. **「▶ 収集開始」** ボタンを押す → 自動スクロールでURLを収集
4. 収集完了（またはストップ）後、エクスポート：
   - **📋 URLコピー** — URL一覧をクリップボードにコピー
   - **📊 TSVコピー** — URL・著者・日時・本文をTSV形式でコピー（Excelに貼付可）
   - **💾 CSV保存** — CSVファイルとしてダウンロード

## 収集データ

各ブックマークから以下の情報を取得します：

| フィールド | 内容 |
|-----------|------|
| URL | ツイートのURL（`https://x.com/user/status/id`） |
| Author | 投稿者のユーザー名 |
| Date | 投稿日時 |
| Text | ツイート本文（先頭200文字） |

## 注意事項

- X のブックマークページ以外では動作しません
- 収集データはブラウザのメモリ上にのみ保持されます（ページをリロードするとクリアされます）
- X の DOM 構造変更により動作しなくなる場合があります
